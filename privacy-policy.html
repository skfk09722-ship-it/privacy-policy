1. 수집하는 개인정보

코순이프로젝트(이하 "앱")는 Threads API를 통해 다음 정보를 처리합니다:

  • 공개된 Threads 게시물 텍스트 및 반응 데이터 (좋아요, 댓글 수)
  • 공개된 Threads 사용자 프로필 정보 (사용자명, 팔로워 수)

2. 개인정보의 이용 목적

  • 콘텐츠 트렌드 분석 및 바이럴 콘텐츠 탐지
  • 게시물 성과 분석 및 콘텐츠 전략 개선
  • 자동화된 콘텐츠 게시 및 관리

3. 개인정보의 보관 및 파기

수집된 데이터는 로컬 데이터베이스에 저장되며, 30일이 경과한 비바이럴 데이터는 자동 삭제됩니다. 사용자가 앱 사용을 중단할 경우 모든 데이터를 즉시 삭제합니다.

4. 개인정보의 제3자 제공

수집된 개인정보는 제3자에게 제공하지 않습니다. 모든 데이터는 앱 운영자의 로컬 환경에서만 처리됩니다.

5. 데이터 보안

API 액세스 토큰은 환경변수로 안전하게 관리되며, 데이터베이스는 로컬 환경에서만 접근 가능합니다.

6. 사용자 권리

사용자는 언제든지 자신의 데이터에 대한 열람, 수정, 삭제를 요청할 수 있습니다.
